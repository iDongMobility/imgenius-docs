人员总览列出了人员相关的总览信息，通过筛选器选择用户让其显示在页面上后，可以看到该用户当前待处理作业组数、已处理作业组数和发布的日志数的简单信息，点击即可进入该用户总览信息页面。

* 选择一个用户点击进入总览信息页面，就可以看的该用户相关的信息。
  * 概要信息
    * 基本信息：包括用户的角色、登录名、手机号码、邮件地址和配置的用户属性。
    * 最新动态：用户有关的最新的动态。
    * 排班表：用户所属的排班表。
    * 待处理作业组：待用户去进行处理的作业组。
    * 已处理作业组：用户参与的已经处理的作业组。
    * 发布的日志：用户发布过的日志。
