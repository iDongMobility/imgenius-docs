## imgenius需要卸载重装操作步骤

1. 撤销部署, 停止IIS网站,关闭MongDb服务，此时系统停用。  
2. 备份数据库和MongDb。
3. 将备份文件转移到自己电脑内。
4. 记录EOC相应自定义导航设定方式。三种链接方式。  
5. 卸载IMgenius软件。删除C:\Program Files (x86)\iDong  
6. 安装新版本IMgenius软件。  
7. 开启IIS，im部署工具重新部署。将步骤4文件放入对应位置。  
8. 开启MongDb服务，打开SMC数据库自动升级。  
9. 重新部署SMC。  
10. 登录EOC确定数据无异常，归档作业组有数据
